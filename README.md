🔊 VoiceCraft – Text to Speech Converter

Transforming Text into Natural Voice with Web Technology

🚀 Overview

VoiceCraft is a browser-based Text-to-Speech (TTS) application that converts written text into spoken words using the Web Speech Synthesis API.

This project highlights how powerful features can be built using pure front-end technologies — without any backend — while delivering a smooth and interactive user experience.

✨ Features
📝 Text to Voice Conversion
Instantly convert typed text into speech output

🎙️ Voice Selection
Choose from multiple voice options such as:
Microsoft David (English - US)
Microsoft Mark (English - US)
Microsoft Zira (English - US)
⏯️ Playback Controls
▶️ Play
⏸️ Pause
⏹️ Stop

🎨 Modern UI Design
Beautiful gradient background (teal → purple) for an engaging look

📱 Responsive Input Area
User-friendly textarea that adapts across devices

⚡ Client-Side Only
Fully functional without any backend using the browser’s native API

🛠️ Tech Stack
HTML5 – Structure
CSS3 – Styling and gradient UI
JavaScript (Vanilla JS) – Functionality and controls
Web Speech Synthesis API – Core speech engine

💡 Purpose of This Project
This project demonstrates:
Integration of browser-native APIs
Handling real-time user interactions
Building accessible and assistive technology features
Creating functional apps without backend dependencies
